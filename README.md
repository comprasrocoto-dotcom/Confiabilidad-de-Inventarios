# Sistema de Auditoría de Inventarios

## Actualizar la base de datos

1. Ir a GitLab → carpeta **`public/`**
2. Seleccionar **`base-datos-cobros.xlsx`**
3. Clic **Reemplazar** → subir el nuevo archivo
4. Vercel redeploya automáticamente

> El archivo debe llamarse exactamente: `base-datos-cobros.xlsx` (sin espacios, en minúsculas)
